## Project Description
This project contains two beginner-friendly JavaScript applications built to practice core programming concepts. The interactive calculator performs basic math operations, while the student grade tracker stores and processes student grade data using arrays and objects.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## Features
- Interactive calculator with addition, subtraction, multiplication, and division.
- Input validation for empty values and division by zero.
- Student grade tracker that stores student records as objects inside an array.
- Class statistics including total students, average grade, highest grade, and lowest grade.
- Ability to delete individual students and clear all student records.

## How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/mordzy101/iyf-s10-week--04---mordzy101-.git
   ```
2. Open `index.html` in your browser to view the combined project page.
3. You can also open the individual project pages:
   - `calculator/index.html`
   - `gradeTracker/index.html`

## Lessons Learned
While building this project, I learned how to use JavaScript variables, functions, conditionals, arrays, and objects to create interactive web applications. I also practiced reading user input from HTML forms, updating the page with DOM manipulation, and validating user data before processing it.

## Challenges Faced
One challenge was handling invalid input, such as empty calculator fields, division by zero, and grade values outside the 0 to 100 range. I solved this by adding validation checks before running calculations or saving student data.
